关于Hexo & Npm & SAI

资料收集来源于网络，这里只是整合列表，方便查找会不定期的更新列表=￣ω￣=

Hexo

命令 缩写 描述
hexo clean hexo cl 清除缓存
hexo server hexo s 启动本地服务器
hexo server -p hexo s -p 更改访问端口 (默认端口为4000）
hexo server -i hexo s -i 自定义 IP 地址
hexo generate hexo g 生成静态网页
hexo deploy hexo d 将本地数据部署到仓库(如github)
hexo version hexo -v 显示 Hexo 版本
hexo new [layout] <title> hexo n "文章标题" 创建一篇文章
hexo new page xxxx 新建页面
hexo --draft 显示草稿
hexo --silent 隐藏终端信息
hexo --debug 在终端中显示调试信息并记录到 debug.log

Npm

命令 缩写 描述
npm -v npm -v 查看npm版本
npm install xxxx npm -i xxxx 安装xxxx

SAI2达传说

笔的快捷键

命令 描述
A 选区笔
B 喷枪
C 水彩笔
D 软消
E 橡皮擦
F 向下转写
L 套索
M 框选
N 铅笔
S 选区擦
V 画笔
W 魔法棒

Ctrl

命令 描述
Ctrl 移动画布内容
Ctrl + 0 充满视图
Ctrl + B 从剪贴板新建画布（注意是画布不是图层）
Ctrl + D 取消选区
Ctrl + E 向下合并
Ctrl + T 自由变换
Ctrl + S 保存
Ctrl + U 色相 / 饱和度
Ctrl + Z 撤回

Alt

命令 描述
Alt + Delete  填充颜色
Alt + 左键  取色
Alt + 空格 + 左键  旋转画布
Alt + 空格 + 右键  恢复画布旋转

组合按键

命令 描述
Ctrl + Alt + 鼠标左键( 左右拖动 )  调整画笔大小和浓度
Ctrl + Shift  显示当前位置的图层名称
Ctrl + 空格 + 左键单击 放大视图
Ctrl + 空格 + 右键单击 重置视图的放大
Ctrl + Alt+ 空格 + 左键拖拽 视图的变焦框
Ctrl + Alt+ 空格 + 左键单击 缩小视图
Ctrl + Alt+ 空格 + 右键单击 重置视图的缩小

其他

命令 描述
H 水平翻转画布
I 编辑路径
X 切换前景色和背景色
0~9 画笔浓度（一次10%）
[ 选择小一号的画笔大小
] 选择大一号的画笔大小
Tab 显示所有的操作面板
PageUp 以光标位置为中心放大
PageDown 以光标位置为中心缩小
Home 重置视图的显示位置
