色彩的一些调色公式

发表于 2022-06-20
442- 1m
学习笔记-美术色彩

蓝色调:

  • 白色桌面:钛白 + 珍珠白 + 加浅灰蓝 + 那坡里黄 + (少量) 春日青 (推空间加雪青+春日青)
  • 蓝色布:月灰 + 浅蟹灰 +灰色 + 浅灰蓝 + 春日青 + 那坡里黄 (推空间加雪青)

  • 背景:熟褐 (少量) + 雪青 + 春日青 + 月灰

  • 后地板:熟褐 + 浅蟹灰 + 赭石 + 灰色

  • 底架:湖蓝 + 草绿 + 大红 (不调匀直接上)

  • 桌立面:月灰 + 春日青 + 雪青 + 牙黄灰

红色调:

  • 白色桌面:钛白 + 珍珠白 + 加浅灰蓝 + 那坡里黄 (推空间加雪青+春日青)
  • 红色布:肉色 + 豆沙红 + 月灰 + 珍珠白 + 浅灰蓝 + 那坡里黄 (推空间加雪青+春日青)

  • 背景:熟褐 (少量) + 雪青 + 春日青 + 月灰

  • 后地板:熟褐 + 浅蟹灰 + 赭石 + 灰色

  • 底架:湖蓝 + 草绿 + 大红 (不调匀直接上)

  • 桌立面:月灰 + 春日青 + 雪青 + 牙黄灰

黄色调:

  • 白色桌面:钛白 + 珍珠白 + 加浅灰蓝 + 那坡里黄 (推空间加雪青+春日青)
  • 黄色布:月灰 + 嫩黄(加多些) + 米陀 + 珍珠白 + 浅灰蓝 + 那坡里黄 (推空间加雪青+春日青)

  • 背景:熟褐 (少量) + 雪青 + 春日青 + 月灰

  • 后地板:熟褐 + 浅蟹灰 + 赭石 + 灰色

  • 底架:湖蓝 + 草绿 + 大红 (不调匀直接上)

  • 桌立面:月灰 + 春日青 + 雪青 + 牙黄灰

绿色调:

  • 白色桌面:钛白 + 珍珠白 + 加浅灰蓝 + 那坡里黄 + 青苹果 (推空间加雪青+春日青)
  • 绿色布:月灰 + 牙黄灰 + 春日青 + 浅灰蓝 + 那坡里黄 (推空间加雪青)

  • 背景:熟褐 (少量) + 雪青 + 春日青 + 月灰

  • 后地板:熟褐 + 浅蟹灰 + 赭石 + 灰色

  • 底架:湖蓝 + 草绿 + 大红 (不调匀直接上)

  • 桌立面:月灰 + 春日青 + 雪青 + 牙黄灰
